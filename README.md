College Media

🚀 A full-stack social media platform built with modern web and mobile technologies.

This project combines:

🌐 Web application (React + Vite)
📱 Mobile application (React Native + Expo)
⚙️ Backend API & real-time server (Node.js + Express + Socket.IO)

Inspired by platforms like WhatsApp, Instagram, and Threads, the app focuses on real-time communication, social interaction, scalable architecture, and modern UI/UX.

🚀 Getting Started

⚙️ Backend Setup
cd backend
npm install
Create .env
PORT=5000
MONGO_URI=your_database_uri
JWT_SECRET=your_secret
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_key
CLOUDINARY_API_SECRET=your_secret
EMAIL_USER=your_email
EMAIL_PASS=your_password

🌐 Frontend Setup
cd frontend
npm install
npm run dev

Open:

http://localhost:5173
📱 Mobile App Setup
cd mobfrontend
npm install
npm run start

Or:

expo start
